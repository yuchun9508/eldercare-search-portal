# Eldercare Search Portal

This repository was initialized with a minimal Next.js + TypeScript scaffold.

Quick start

1. Install dependencies:

   npm install

2. Run the dev server:

   npm run dev

3. Open http://localhost:3000

What's included

- Next.js app router in `app/`
- TypeScript config (`tsconfig.json`)
- Basic scripts in `package.json`
